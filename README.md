##  Features

### 1. Auth App

* **Toggle System:** Easily switch between Login and Registration modes.
* **Persistence:** Uses `localStorage` to save user credentials securely within the browser.
* **Validation:** Basic checks for required fields and password length.

### 2. GUI Calculator

* **Standard Operations:** Supports addition, subtraction, multiplication, and division.
* **Interactive Interface:** Clean, responsive grid-based design.
* **State Management:** Tracks current inputs and expressions to perform calculations accurately.


## Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Flexbox and Grid layouts for responsive design.
* **JavaScript (Vanilla):** Logic, state management, and DOM manipulation.
* **LocalStorage API:** To handle user account data without a backend server.
